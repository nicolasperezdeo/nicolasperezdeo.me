A modern, responsive portfolio for Nicolas Perez de Olaguer highlighting leadership across machine learning, MLOps, and computer vision for healthcare.

## Tech Stack

- [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/) for subtle motion design

## Getting Started

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:5173`.

## Production Build

```bash
npm run build
```

Static assets are output to `dist/`. Run `npm run postbuild` to copy the production build into `docs/` for GitHub Pages hosting.

## Content

All resume and project information lives in [`src/data/profile.ts`](src/data/profile.ts) for easy updates.
